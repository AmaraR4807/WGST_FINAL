This project explores Intersectionality, Important places and events, Have Queer people always existed? What is LGBTQ? Media representation and suppression. And activism. Each of these topics are essential to anyone curious to learn about LGBTQ+ studies.

LGBTQ+ Studies is an introductory course into the WGSS curriculum. This class focuses briefly on LGBTQ+ topics. Each of the individual topics listed before are significant to the course. This class goes in depth from topics such as history, to contemporary trends, to defining things like pride and what each letter of LGBTQ represents.
